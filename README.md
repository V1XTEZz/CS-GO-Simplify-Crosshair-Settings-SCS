# CS-GO-Simplify-Crosshair-Settings-SCS
Simple way to modify ur crosshair

<br>Download cfg and move here <pre>...\common\csgo legacy\csgo\cfg</pre>

<h3>Settings:</h3>
<blockquote>
  <i>CGap[num]</i> - <code>(5 to -5) Gap between crosshair's lines</code>
  <br><i>CSize[num]</i> - <code>(0 to 10) Size of crosshair [0.5 until 5]</code>
  <br><i>CThickness[num]</i> / <i>CThicc[num]</i> - <code>(1, 2 & 3) Thickness/width of crosshair</code>

  <p>
    <br><i>CStyle[num]</i> - <code>(1 to 5) Changing style of crosshair</code>, 
    <br><i>1 (old crosshair) 2(double crosshair) 3(classic dynamic) 4 (static) 5(shots dynamic only)</i>
  </p>

  <br><i>CDot</i> - <code>turning off/on dot for crosshair (togglable)</code>
  <br><i>CT</i> - <code>turning off/on T style crosshair (togglable)</code>
  <br><i>Outline</i> - <code>turning off/on outline (togglable)</code>
  <br><i>CFF</i> - <code>Friendly warning (togglable)</code>
  <br><i>CGapUWV</i> - <code>Use Weapon Value (togglable)</code>
  <br><i>CCol[num]</i> - <code>(0 to 5) Colors 0(red) 1(green) 2(yellow) 3(blue) 4(cyan) 5(custom)</code>

  <br><i>SCSize[num]</i> - <code>(1 or 2) Sniper crosshair thickness(width)</code></blockquote>
<br>
<h5>Now RGBA</h5>
<p>Unpack .zip file in <code>...\csgo legacy\csgo\cfg</code>
<br>In console write <code>exec EasyRGBA</code></p>
<h3>RGBA Settings:</h3>

<blockquote>
  <b>R</b><i>[num]</i> - <code>Red</code>
  <br><b>G</b><i>[num]</i> - <code>Green</code>
  <br><b>B</b><i>[num]</i> - <code>Blue</code>
  <br><b>A</b><i>[num]</i> - <code>Alpha</code>
  <br>
  <br><i>For each command minimum 0 maximum 255</i>
</blockquote>
<h5>Also available color names here list of colors</h5>
<blockquote>
  <b>CWhite</b> <code>(R 255 ; G 255 ; B 255)</code>
  <br><b>CBlack</b> <code>(R 0 ; G 0 ; B 0)</code>
  <br><b>CRed</b> <code>(R 255 ; G 0 ; B 0)</code>
  <br><b>CGreen</b> <code>(R 0 ; G 255 ; B 0)</code>
  <br><b>CBlue</b> <code>(R 0 ; G 0 ; B 255)</code>
  <br><b>CCyan</b> <code>(R 0 ; G 255 ; B 255)</code>
  <br><b>CYellow</b> <code>(R 255 ; G 255 ; B 0)</code>
  <br><b>CPink</b> <code>(R 255 ; G 0 ; B 255)</code>
  <br><b>CPurple</b> <code>(R 120 ; G 0 ; B 255)</code>
  <br><b>COrange</b> <code>(R 255 ; G 120 ; B 0)</code>
</blockquote>
<h3>Settings for Style 2</h3>
<blockquote>
  <b>CIA</b> - <code>Inner alpha (0 to 1, +0.1 only)</code>
  <br><b>COA</b> - <code>Outer alpha (0 to 1, +0.1 only)</code>
</blockquote>
